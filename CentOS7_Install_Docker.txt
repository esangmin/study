1. Docker 기본 설치 (1안)
 - 이번 버전 제거
   . sudo yum remove docker \
                  docker-client \
                  docker-client-latest \
                  docker-common \
                  docker-latest \
                  docker-latest-logrotate \
                  docker-logrotate \
                  docker-selinux \
                  docker-engine-selinux \
                  docker-engine
 - 최신 저장소 설정   
   . yum install -y yum-utils device-mapper-persistent-data lvm2
   . yum-config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo
 - Docker 최신 버전 설치
   . yum install docker-ce

1. Docker 기본 설치 (2안)
 - yum 패키지 업데이트
   . yum -y update

 - Docker & Docker Registry 설치
   . yum -y install docker docker-registry


1. Docker 기본 설치 (선택)
 - sudo yum -y update
 - 이미 모두 설치되어 있음 : 실행 불필요 sudo yum install -y yum-utils device-mapper-persistent-data lvm2
 - sudo yum-config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo
 - sudo yum install docker-ce

2. Docker 실행 및 자동(On Boot) 실행 서비스 등록
 - 부팅시에 실행하도록(systemctl)에 등록
   . sudo systemctl enable docker.service
 - Docker 실행
   . sudo systemctl start docker.service
 - systemctl 사용 Docker Status 확인
   . sudo systemctl status docker.service
3. Docker 이미지 다운로드 (1안)
 - Docker 사용 준비 완료 후 이미지 다운로드
   . sudo docker pull centos
 - Docker 컨테이너 실행 (강제 종료 : Ctrl-p + Ctrl-q)
   . sudo docker run -i -t centos /bin/bash

3. Docker 이미지 다운로드 (2안)
 - sudo docker run --rm hello-world
 
 
4. 특정유저 docker 그룹에 귀속 
 - docker는 기본적으로 root권한이 필요함. root가 아닌 사용자가 sudo 없이 사용하려면
   해당 사용자를 docker 그룹에 추가해야 함
 - 이미 그룹이 있음 sudo groupadd docker
 - sudo usermod -aG docker $USER
