○ 개요
 - 엔진엑스(Nginx)는 Igor Sysoev라는 러시아 개발자가 동시접속 처리에 특화된 웹 서버 프로그램이다. 
 - Apache보다 동작이 단순하고, 전달자 역할만 하기 때문에 동시접속 처리에 특화되어 있다.

○ 설치 on Windows
 - http://nginx.org/en/download.html
 - 압축 해체
 - nginx.exe 실행
 - 명령어
   . nginx.exe -s stop --> 강제 종료
   . nginx.exe -s quit --> 접속한 사용자의 처리가 다 끝난 후 종료
   . nginx.exe -s reload --> 설정파일 갱신
   . nginx.exe -s reopen --> 로그파일 갱신
 - 방화벽에 nginx 등록
   . 제어판 > Windows 방화벽 > Windows 방화벽을 통해 앱 또는 기능 허용
   . 다른 앱 허용 버튼 클릭 후 nginx.exe 등록 : Nginx에서 사용하는 포트는 자동으로 열림
 - 윈도우 서비스에 nginx 등록
   . https://github.com/lkinsella/nginx-svc 프로젝트 소스 다운로드
   . 빌드 
   . bin/Realse/nginxsvc.exe.config 수정
   . 윈도우서비스 등록 : nginxsvc.exe /install
   . 윈도우서비스 제거 : nginxsvc.exe /uninstall
     
○  React 앱 배포 on Windows
 - npm run build
 - nginx설치폴더/conf/nginx.conf 수정
   . root : html; --> 빌드된 build 폴더 (\는 /로 변경)
   . 제어판 > 서비스 에서 nginx 다시 시작
 - 웹브라우저에서 해당 url 입력 후 확인
